ARDUINO CODE FOR CBEMC-4 MCO1 PROJECT BY TERMINAC

Timer 1 = Alarm every second prints BPM.
Interrupt 1 = Prints out Avg BPM data. If there is no finger on heart sensor, Avg BPM is set to 0. Also prints whether Movement is Detected or Undetected.
Timer 2 = Alarm every minute to print time elapsed in minutes
Interrupt 2 = prints "[Time Elapsed: X minutes]"
